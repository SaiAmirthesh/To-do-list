# Todo List App

A simple and elegant todo list application built with React, Vite, and Supabase.

Live Link : ![To-do-App](https://to-do-list-sand-five-94.vercel.app/)  

## Features

- User authentication (Sign up / Sign in)
- Create, read, update, and delete todos
- User profiles with customizable details
- Beautiful rose-themed UI
- Responsive design

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (Database + Authentication)
- **Icons**: Custom emoji icons
